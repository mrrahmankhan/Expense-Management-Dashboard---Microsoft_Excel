# Expense Management Dashboard (Excel)

## Project Overview
An interactive Excel dashboard built using Pivot Tables, Pivot Charts, Slicers, and KPI Cards to analyze organizational expenses and budget utilization.

## Features
- Interactive filtering using Slicers
- KPI Cards
- Department-wise expense analysis
- Category-wise expense analysis
- Monthly trend analysis
- Budget vs Actual comparison
- Payment method analysis
- Approval rate tracking

## Dashboard KPIs
- Total Expenses
- Budget Allocated
- Budget Remaining
- Total Transactions
- Approval Rate
- Average Expense

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- GETPIVOTDATA
- Dashboard Design

## Insights
- Highest Spending Department: HR
- Highest Spending Category: Software
- Budget Utilization Tracking
- Expense Trend Monitoring
